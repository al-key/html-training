# html-training

## 学習メモ：GitHub の基本概念(AIまとめ)

### リポジトリ（Repository）
- プロジェクトのすべてのファイルと変更履歴を保存する場所
- この `html-training` がリポジトリ
- リポジトリ内には、ファイル、ブランチ、プルリクエストなどが含まれる

### ブランチ（Branch）
- プロジェクトの作業用の分岐
- メインの流れに影響を与えないように別の道筋で作業できる
- 例：`main`（メイン）、`al-key-patch-1`（今回の作業用）

### プルリクエスト（Pull Request）
- あるブランチの変更を別のブランチに反映させることを提案する機能
- `al-key-patch-1 → main`：al-key-patch-1 での変更を main に取り込む提案
- レビュー、承認、マージの流れがある

### README.md
- リポジトリの説明書・案内書
- プロジェクトにアクセスすると最初に表示される
- 使い方やプロジェクト説明を記録する場所
